# Advanced Voting Machine Simulation
A demo app for secure voting using Aadhaar, live face verification, and blockchain.

## Features
- Add voters with face enrollment (upload or live capture).
- Enroll/update faces.
- Cast votes with live face verification.
- View blockchain ledger.

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `streamlit run app.py`
